# Diaspora For Typecho

### 网站图标相关

网站需要 3 个 logo 图片，分别是：

- `assets/images/logo.png` (124x52) 网站首页 logo
- `assets/images/logo_black.png` (124x52) 网站弹出菜单时候的深色 logo
- `assets/images/logo_min.png` (48x48) 网站文章详细页面的小 logo
